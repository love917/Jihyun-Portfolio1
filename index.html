Parallax Effect

이를 통해 웹페이지에 깊이와 생동감을 더해 사용자에게 다채로운 시각적 경험을 제공할 수 있습니다.

Demonslayer

CSS와 JavaScript를 활용하여 사용자가 스크롤을 내릴 때 이미지가 동적으로 움직이도록 구현했습니다. 이렇게 함으로써 웹페이지에 재미를 더하고, 사용자가 더 오래 머무를 수 있도록 효과를 주었습니다.

Voyalabs

웹사이트 'Voyalabs'는 JavaScript로 동적인 커서 모양을 구현하고 lottiefiles 애니메이션을 사용해 시각적으로 흥미로운 정보를 제공하며,
특정 영역에 도달하면 배경 테마가 바뀌는 효과를 적용했습니다.
이러한 요소들을 통해 사용자 경험을 향상시킬 수 있습니다.

Sushi

부여하여 사용자에게 흥미로운 경험을 줄 수 있습니다.

Musign

메뉴 화면이 자연스럽게 나타나도록 했습니다. SVG로 직접 제작한 원형 애니메이션이 특정 영역에 도달하면 실행되며, CSS 변수와 J-Query를 활용하여 테마 색상도 동적으로 변경할 수 있습니다.

Responsive

이러한 과정을 통해 반응형 페이지를 만드는 방법을 배웠으며 기본 웹 기술을 활용해 세련된 디자인과 유용한 기능을 제공하는 웹사이트를 완성했습니다.
